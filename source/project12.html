Task Manager

Самостоятельный проект.

Решил, как и многие другие, написать свое приложение для ведения списка задач. Для backend'а я исользовал фреймворк Express. Для frontend'а - React на функциональных компонентах и хуки.

На данный момент готово самое основное - сервер круиться на Heroku и рабочий веб-интерфейс, который получает, отправляет данные с сервера и на сервер, похволяет создавать, редактировать, удалять задачи. Сейчас пишу авторизацию для пользователей. В планах - создание подзадач и меток.

В процессе разработки использовалось и изучалось следующее:

  • Ajax, REST
  • React + Hooks
  • Express
  • React Router
  • Деплой (Heroku).
